## Currently Building

### [SSC Mentor / SSC GK Score Booster](https://tinyurl.com/ssc-gk-score-booster)

A mobile-first quiz and revision platform for SSC aspirants.

It helps students practice GK questions, review mistakes, save important questions, track quiz history, and get AI-supported explanations.

**Live App:** [Try SSC GK Score Booster](https://tinyurl.com/ssc-gk-score-booster)  
**Repository:** [View GitHub Repo](https://github.com/until-aman/SSC-GK-SCORE-BOOSTER-V2)

### Current product focus

- Personalized revision plans
- AI mentor for weak-topic guidance
- Quiz history and saved-question flows
- Performance analysis for aspirants
- API optimization and cleaner app architecture
- Consistent mobile-first UI across the product
